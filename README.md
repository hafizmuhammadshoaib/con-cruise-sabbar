## Docker

```bash
$ docker-compose up
```

## Installation

```bash
$ npm install
```

## Migrations

```
$ npm run typeorm:run
```

## Running the app

```bash
$ npm run start:dev
```

## Running the CLI

```bash
$ npm run build # for first time
$ npm run cli <command>
$ npm run cli help # to see the list of commands
```

## Running Integration Tests

```bash
$ npm run test:e2e
```
